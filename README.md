# CalcPE_UFABC

Compile Calc.c and the custom dataStructures library using gcc and run with the following line:
```
gcc -Wall Calc.c dataStructures.c -o CalcExecApp
```
To compile use GCC, it was tested on Windows 11 and UNIX like system as MacOS;
As seen before at the command line above, it has a custom library that needs to be compiled along the main application.
